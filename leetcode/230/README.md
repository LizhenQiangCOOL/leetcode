# [230. 二叉搜索树中第K小的元素](https://leetcode.cn/problems/kth-smallest-element-in-a-bst/)

## 1. 题目描述
给定一个二叉搜索树的根节点 root， 和一个整数 k，请你设计一个算法查找其中第 k 个最小的元素（从1开始计数）


## 2. 示例
```
    3
   / \
  1   4
   \
    2
输入： root = [3,1,4,null,2] k = 1
输出：1
```
```
              5
             / \
            3   6
           / \
          2   4
         /
        1
输入： root = [5,3,6,2,4,null,null,1] k=3
输出：3
```
